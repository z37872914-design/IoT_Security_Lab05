Project Description

This repository contains my solution for Lab Assignment No. 05: IoT and Cyber-Physical Systems Security.
The notebook shows how to:

Encrypt the data using AES (lightweight cryptography)

Decrypt the data on the server side

Simulate the IoT security lifecycle (5 stages)

Demonstrate a simple secure boot verification using firmware hash
